## Key Binds
- **YOU WILL FIND ALL KEYBINDS IN hypr/binds.conf and for mangowc mango/binds.conf**
- Some Imp KEYBINDS 

| Keybind | Action |
|--------|--------|
| Super + ` | Open terminal | 
| Super + Q | Close window | 
| Super + A | App launcher | 
| Super + L | Wallpaper Switcher | 
| Super + B | Waybar Switcher |
| Super + Shift + S | Screenshot |
